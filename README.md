# Casa Provisioner

A custom integration for Home Assistant.

## Installation via HACS

1. Make sure you have [HACS](https://hacs.xyz/) installed in your Home Assistant instance.
2. In the Home Assistant UI, go to **HACS** > **Integrations**.
3. Click the three dots in the top right corner and select **Custom repositories**.
4. Add the URL of this repository and select **Integration** as the category.
5. Search for "Casa" in HACS and click **Download**.
6. Restart Home Assistant.
7. Go to **Settings** > **Devices & Services** > **Add Integration** and search for "Casa" to configure it.
